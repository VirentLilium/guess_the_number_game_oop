# 🎮 Guess The Number Game (OOP)

Учебный проект на Python, посвящённый переводу консольной игры «Угадай число» с функционального подхода на объектно-ориентированное программирование.

Проект был создан для практики работы с классами, методами, состоянием объекта, декораторами и валидацией пользовательского ввода.

## Цель проекта

В проекте отрабатывались:

* создание и использование классов;
* методы экземпляра и статические методы;
* работа с состоянием объекта;
* декораторы;
* валидация пользовательского ввода;
* организация программы с использованием ООП.

## Возможности игры

* угадывание случайного числа;
* подсказки при неверном ответе;
* расчёт оптимального количества попыток;
* повторный запуск игры без перезапуска программы;
* специальные команды для администратора:

  * просмотр статистики;
  * просмотр правильного ответа.

## ⚙️ Установка проекта

### Клонирование репозитория

```bash
git clone https://github.com/VirentLilium/guess_the_number_game_oop.git
cd guess_the_number_game_oop
```

### Создание виртуального окружения

```bash
python -m venv venv
```

### Активация виртуального окружения

Windows:

```bash
venv\Scripts\activate
```

Linux / macOS:

```bash
source venv/bin/activate
```

### Обновление pip и установка зависимостей

```bash
python -m pip install --upgrade pip
pip install -r requirements.txt
```

## Запуск игры

```bash
python guess_number.py
```

## Запуск тестов

```bash
pytest
```

## Режим администратора

Для входа в режим администратора используйте имя:

```text
Admin
```

После входа становятся доступны дополнительные команды:

```text
stat
```

Показать статистику игры.

```text
answer
```

Показать загаданное число.
